SBA 7(a) Loan Repayment Prediction

Full Project Report: 
Overview:

This project builds a machine learning pipeline to predict loan repayment outcomes for small businesses using SBA 7(a) loan data. The work focuses on large-scale data processing, model performance, and interpretability.

The dataset includes 1.2M+ loan records, enabling realistic financial risk modeling.

High Level Steps:

Cleaned and processed 1.2M+ loan records

Performed EDA to understand borrower and loan patterns

Engineered financial and borrower-level features

Trained and compared multiple models:

Logistic Regression

Random Forest

XGBoost

CatBoost

Analyzed feature importance to identify key drivers of repayment and default risk

Programmed Using:

Python (pandas, NumPy, scikit-learn)
